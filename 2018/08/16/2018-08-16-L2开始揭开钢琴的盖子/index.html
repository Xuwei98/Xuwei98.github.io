冯诺依曼思想

  1. 存储程序的主要思想：将程序和数据存放到计算机内部的存储器中，计算机在程序的控制下一步一步进行处理
  2. 计算机由五大部件组成：输入设备、输出设备、存储器、运算器、控制器
  • 程序存到内存里，用IP指针指向它，进行取址执行（自动循环）。

x86 PC开机

  • ROM BIOS是固化的，固化在0xFFFF0中。
  1. x86PC刚开机时属于实模式。
  2. 将CS = 0XFFFF；IP = 0X0000;
  3. 执行BIOS内部代码。
  4. 检查ROM，键盘， 显示器， 软硬磁盘， 主板等。

  5. 将磁盘0磁道0扇区读入0X7c00处。（一个扇区大小为521B）。

    • 0磁道0扇区为引导扇区(启动设备的第一个扇区)，是第一个扇区。

    • 第一个扇区存放着开机后执行的第一段我们可以控制的程序。

  6. 设置CS = 0X07c0；IP = 0X0000。

    引导扇区代码：bootsect.s
