操作系统基础 L1 什么是操作系统

发表于 2018-08-16 | 分类于 操作系统 | |
字数统计: 218 | 阅读时长 ≈ 1

课程重点总结

  1. 操作系统将编程语言与计算机硬件进行联系。
  2. 操作系统是计算机硬件与应用之间的一层软件。
  3. 管理硬件部分：
    • CPU
    • 内存
    • 终端
    • 磁盘
    • 文件
    • 非重点（网络，电源， 多核）

学习层次

  • 懂得接口使用。
  • 懂得接口的内部实现。
    • 学会改动操作系统。
    • 手动编写操作系统。
  • 从板子上搭建一个操作系统。

课程目标

  1. 理解真实操作系统的运转。
  2. 做好8个实验，地址。
  3. 学会增加模块，修改系统。

斯坦福的四个实验

  1. 扩展线程: 实现线程调度
  2. 实现系统调用: 将整个接口剥掉，添加
  3. 实现虚存管理：扩展实现内存管理
  4. 扩展文件系统：扩展实现一个文件管理

Learn OS concepts by conding them!
