c++要点复习

封装

将数据和操作数据的函数衔接在一起，使程序结构紧凑，提高内部数据安全性

继承

增加软件可扩充性和代码的可重用性

多态

对问题进行更好的抽象，有利于代码的维护和可重用

标准输入输出流

输入输出流类的对象，
cin, cout ,cerr
cin istream输入流对象
cout ostream输出流对象
cin, cout 能自动给调整输入输出格式。
endl表示“\n”换行

结构体与类

class中的访问控制

public 共有的，无访问限制
private 私有的只能自己访问
protected 父类子类中的访问限制
strcut 默认public
class 默认private

类与对象

对象是实例化的类

构造函数与析构函数

函数的重载

相同：返回值，函数名
不相同：参数类型，参数个数不同

void output(int a,int b = 5);
void output(int a)
带有缺省值的函数也无法构成函数重载

this指针

1.隐含的指针，指向对象本身，代表对象的地址
2.所有对成员变量的访问 隐含加上前缀this->。
3.如果成员函数的参数与成员变量名重复，使用this指针进行区分。

继承补充


在声明子类对象后，父类构造函数->子类构造函数。
生命周期结束后，子类析构函数->父类构造函数。

函数的覆盖

1.发生在父类与子类之间，子类函数默认覆盖同名父类函数。
2.在父类不再适用子类，可用函数覆盖实现函数重写。
3.如果需要同时使用父类函数，通过作用域符号’ :: ‘实现。

c++的多态性

将会调用父类的breath函数，这里存在着隐式的强制转换
pan = (animal *)&fh;

animal、fish的内存地址开头是一样的，所以会调用animal的breath。
如果想要调用fish的breath,采用虚函数

多态

c++编译器在编译的时候发现animal的breath()函数是虚函数，所以采用迟邦定，在运行的时候依据对象类型确认调用哪一个函数。

纯虚函数

例如

这个类叫做抽象类,不能直接实例化对象。
纯虚函数没有具体操作内容，只有操作名称，让派生类继承去实现具体内容。
如果基类不太确定方法但又是必须的，可以使用纯虚函数。

引用

引用相当于变量的别名，并且在声明的时候就要初始化。
引用和指针的内存模型

引用能减少值的拷贝

tips

声明虚函数加virtual，但在定义时不加virtual
// #include “” 首先从当前目录查找，最后还是会搜索系统目录，一般自定义头文件
// #include <> 从系统目录开始查找，系统头文件
通过#prama once //避免重复定义
也可采用预编译指令符

2.编译各个cpp文件 ，生成.obj文件
